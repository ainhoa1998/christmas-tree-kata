# Kata christmas tree

## Enunciado de la kata

Haz una función `christmasTree` que reciba un número que sea la altura de árbol. Debe devolver un string con el árbol de navidad dibujado con ASCII.

Ejemplos:

`christmasTree(1)`

```
*
```

`christmasTree(2)`

```
 *
***
```

`christmasTree(3)`

```
  *
 ***
*****
```
